**Website:** https://dns-dex.com/

## Project Overview

DNS DEX is a revolutionary decentralized exchange for domain name services that combines traditional DNS infrastructure with blockchain technology. The platform enables users to trade, manage, and monetize domain names while providing decentralized DNS resolution services.

## Key Features

- **Decentralized DNS Resolution**: Blockchain-based domain name resolution
- **Domain Name Trading**: Peer-to-peer marketplace for domain transactions
- **Smart DNS Contracts**: Automated domain management and renewal
- **Censorship Resistance**: Unstoppable domain resolution and hosting
- **Revenue Sharing**: Domain owners earn from DNS queries and traffic
- **Token Governance**: $DNSD token for platform governance and rewards

## Technology Stack

- **Blockchain**: Ethereum with custom DNS smart contracts
- **Frontend**: React.js with Web3 integration for decentralized access
- **Backend**: Node.js with DNS server implementation
- **Database**: IPFS for distributed domain record storage
- **DNS Protocol**: Custom DNS resolver with blockchain integration
- **P2P Network**: Distributed network for DNS query resolution
