# Libertas Coffee

**Website:** https://www.libertascoffee.store/

## Project Overview

Libertas Coffee is a premium coffee shop that combines exceptional coffee with blockchain innovation. They offer a unique revenue-sharing coffee drop-shipping model with automatic token rewards.

## Key Features

- **Premium Coffee Experience**: Artisan-crafted specialty beverages made with ethically-sourced beans
- **$BREW Token Rewards**: Customers earn $BREW tokens with every purchase that can be traded for discounts, exclusive items, or real value
- **HandCash Integration**: Seamless payment processing with automatic reward distribution
- **Revenue-Sharing Drop-Shipping**: $BREW token holders earn dividends from every sale at point of sale
